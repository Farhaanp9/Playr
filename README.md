## What is Playr?

Playr transforms natural language descriptions into fully functional HTML5 games. Describe the game you want, and Playr's AI pipeline handles everything — game logic, assets, bundling — and delivers a playable game in seconds. Browse and play other users' creations through a TikTok-style infinite feed.

---

## Features

- **AI Game Generation** — Describe a game in plain English and get a playable HTML5 game in seconds
- **Multi-Model LLM Pipeline** — Orchestrates multiple AI models for game design, code generation, and asset creation
- **Instant Play Feed** — TikTok-style infinite scroll of user-generated games, playable on mobile and web
- **Real-Time Progress** — Live streaming updates as your game is being generated
- **10 Game Types** — Dodge, flappy, runner, jumper, stacker, snake, tetris, breakout, quiz, and fully custom games
- **Optimized Performance** — 85% client memory reduction through asset optimization and efficient bundling
